# 🚀 Personal Portfolio Website

A modern, interactive, and fully responsive personal portfolio website built using **HTML, CSS, and JavaScript**.

This portfolio showcases professional skills, projects, education, and contact information through a visually engaging user experience featuring custom animations, particle effects, aurora backgrounds, and smooth scrolling interactions.

---

## 🌐 Live Demo

🔗 https://portfolio-51e24.web.app/

---

## ✨ Features

### 🎨 Modern UI Design
- Dark futuristic theme
- Aurora glow effects
- Glassmorphism-inspired navigation
- Responsive layout for desktop and mobile

### 🖱️ Interactive Experience
- Custom animated cursor
- Comet trail effect
- Hover animations
- Smooth scrolling navigation

### 🌌 Dynamic Background
- Particle system
- Shooting stars animation
- Floating geometric shapes
- Animated gradient glows

### 📱 Fully Responsive
- Mobile-friendly layout
- Adaptive navigation menu
- Optimized spacing and typography

### 📂 Portfolio Sections
- Hero Section
- About Me
- Skills
- Projects
- Education
- Contact Information

### ⚡ Performance Enhancements
- Lightweight implementation
- No external frameworks
- Smooth animations using vanilla JavaScript
